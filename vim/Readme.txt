01.对于Linux,"plugin"及"doc"需要放置到"~/.vim"中。
   对于Windows,放置到"D:\Program Files (x86)\Vim\vimfiles"。
02.Linux下的ctags下载:apt-get install exuberant-ctags。
   ctas网址:http://ctags.sourceforge.net/,https://sourceforge.net/projects/ctags/files/。
03.各种插件的简介及下载地址：
--a.Conque Shell : Run interactive commands inside a Vim buffer 
   https://www.vim.org/scripts/script.php?script_id=2771
--b.Source Explorer (SrcExpl):A Source code Explorer for viewing definition(s) and contextual info 
   https://www.vim.org/scripts/script.php?script_id=2179
--c.grep.vim : Search tools (grep, ripgrep, ack, ag, findstr, git grep) integration with Vim 
   https://www.vim.org/scripts/script.php?script_id=311
